## ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que se utiliza para dar formato a texto de manera sencilla y legible. Su principal objetivo es permitir que el contenido pueda escribirse de forma fácil y que, al mismo tiempo, sea convertido a HTML u otros formatos sin complicaciones.

Se usa comúnmente para:
- Documentación de proyectos (como archivos README.md)
- Blogs y plataformas de contenido
- Foros y aplicaciones de mensajería
- Notas técnicas y apuntes

## Formatos comunes en Markdown

- **Encabezados:** Se crean usando el símbolo `#`
- **Negritas:** `**texto**`
- *Cursivas:* `*texto*`
- Listas:
  - Listas ordenadas
  - Listas no ordenadas
- **Enlaces:** `[texto](URL)`
- **Imágenes:** `![texto alternativo](URL)`
- **Código:** 
  - En línea: `código`
  - Bloques de código usando triple acento grave

Markdown es muy popular porque es fácil de aprender, rápido de escribir y compatible con muchas plataformas.
